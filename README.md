# MiLanding

## Descripción
MiLanding es un e-commerce desarrollado en React, diseñado para ofrecer una experiencia intuitiva en la búsqueda de servicios de recursos humanos. Este proyecto permite a los usuarios navegar entre diferentes servicios, filtrarlos por categorías y gestionar un carrito de compras. Utiliza Firebase Firestore para almacenar y recuperar los datos de los servicios ofrecidos.

## Características
- Navegación fluida entre diferentes vistas.
- Filtrado de servicios por categorías (Servicios generales y Servicios personalizados).
- Integración con Firebase Firestore para la gestión de datos.
- Interfaz responsiva utilizando Bootstrap.
- Indicador de carga (DotLoader) mientras se obtienen los datos.

## Tecnologías Usadas
- **React**: Biblioteca para construir interfaces de usuario.
- **Firebase Firestore**: Base de datos en tiempo real para el almacenamiento de datos.
- **Bootstrap**: Framework CSS para el diseño responsivo.
- **React Router**: Manejo de rutas en la aplicación.
- **DotLoader**: Componente de carga para mejorar la experiencia del usuario.

## Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/karengi4/MiLanding.git
2. Navega al directorio del proyecto:
   ```bash
   cd MiLanding
4. Instala las dependencias:
   ```bash
   npm install
5. Ejecuta la aplicación:
   ```bash
   npm start

## Uso
Accede a la aplicación en tu navegador en http://localhost:3000. Explora los servicios, selecciona categorías y gestiona tu carrito de compras para realizar consultas.

## Estructura del Proyecto
src/: Contiene todos los archivos fuente de la aplicación.
   * components/: Componentes reutilizables como BusquedaItem, Item, etc.
   * context/: Contexto para la gestión del carrito de compras.
   * pages/: Páginas de la aplicación.
   * services/: Servicios para la integración con Firebase.
